特色软件

特色软件的功能让人眼前一亮，但又相对小众

Everything

文件快速搜索工具

Everything 以极快的速度搜索 NTFS 格式硬盘中的文件名，它能在几秒钟内从数百GB的硬盘中精确搜索到需要的内容，支持高级搜索、正则表达式、网络搜索、离线搜索等等。

Skitch

漂亮的图片标注、标记工具

Skitch 通过简单的拖拽，就能在电脑或者手机上的图片上做出漂亮的标记，如画个圆圈，做个注释，画个箭头，标出精彩之处。支持 Win/Mac、iOS/Android。

文本编辑器

文本编辑器是码农的标准配置，比起文档编辑，一款顺手的编辑器能无限制提高效率。(名单来自这里)

Notepad++

Notepad++：多次获得SourceForge社区最佳开发工具奖，开源软件，知名度极高，来自台湾，如果你还没用过，推荐尝试。

Vim

Vim：Unix 下最受欢迎的文本编辑器之一，命令行操作，入门稍显困难，但这是一款学会后可以改变你对软件看法的软件。

Sublime Text 3

Sublime Text 3：最初设计为 Vim 提供多功能扩展，后发展为跨平台编辑器，支持自定义及可通过插件扩展功能，快速且顺畅。

EditPlus

EditPlus：一款很好的替代 Windows 记事本与写字板的工具，拥有很多适用于网页设计师及程序员的功能和特性。

Emacs：与 Vim 一起受到 Unix 用户喜爱，具有极强的扩展性，而 Vim 与 Emacs 的优劣一直伴随着争论。

PDF 阅读器

PDF 是一种跨平台、能保留文件原有样式、开放标准的文本格式，已成为 ISO 32000 国际标准。(名单来自这里)

福昕阅读器

福昕阅读器：在全球拥有大量用户，最优秀的国产软件之一。Ribbon 界面，支持手写签名、插入印章等。

Sumatra PDF

Sumatra PDF：是一款开源的 PDF 阅读器, 小巧、迅速是其最大的特点，支持快捷键与命令行参数，如果你仅仅有阅读需求，这是不错的选择

PDF-XChange

PDF-XChange Viewer：全能 PDF 阅读工具，在加载高分辨率扫描文件时速度优异，渲染效果出众，显示更加细腻。需付费使用，Lite 版本针对非商业用途免费。

Adobe Reader

Adobe Reader：PDF 格式的开发母公司 Adobe 出品，支持广泛，被绝大多数三方厂家推荐为默认 PDF 阅读器，用户众多，体积略显庞大。

Google Chrome

Chrome：浏览器内置可阅读 PDF 文档让在线查看 PDF 变得更加简单，你无需下载文档就可以直接阅读，功能略显简陋，却是临时预览 PDF 首选。

笔记记录工具

现代人最常使用的记录方式是纸和笔，但互联网数字化带来的更加方便的记录方式，可以不用担心容量、安全、丢失等等问题，易分享、易记录，除了手感摸起来没有纸好，能数字化还是要趁早。(名单来自这里)

印象笔记

印象笔记：老牌跨平台笔记工具，国际版 Evernote。一家立志于做百年公司的企业，安全、可靠。

为知笔记

为知笔记：越来越好的笔记应用，记录、查阅一切有价值的信息，同样跨平台支持。

OneNote

OneNote：微软 Office 组件之一，功能非常强大的个人信息整合与多人协作工具，与 Office 各组件完美整合。

有道云笔记

有道云笔记：网易旗下笔记工具，同样跨主流平台支持，文字、手写、录音、拍照多种记录方式，支持任意附件格式。

Google Keep

Google keep：唯一轻量级笔记应用，适合于临时记录，仅有 Android 客户端，其他平台需要使用网页版。

电子邮件客户端

电子邮件是最古老的互联网应用之一，但直到今天作用还举足轻重，挑选一款适合自己的客户端，你会喜欢上收发邮件的感觉。(名单来自这里)

Foxmail

Foxmail：从 1997 年就有的邮件客户端，目前已被腾讯收购，老牌、优秀，功能强大。

Outlook

Outlook：大名鼎鼎的 Microsoft Office 组件之一，除了电子邮件，还包含了日历、任务管理、联系人、记事本等功能。

Thunderbird

Thunderbird：知名开源邮件客户端，与 Firefox 出自同门 Mozilla，有强大的扩展能力。

网易邮箱大师

网易邮箱大师：网易旗下电子邮件客户端，除与网易邮箱配套，还支持支持163/126/QQ/Gmail/企业邮箱等各类邮箱域名

Gmail

Gmail：功能上可以称为业界标杆，用户数量世界第一，或许你真的找不到比它更好的邮件系统。

网页浏览器

网页浏览器是互联网的基础，在桌面端，人们越来越依赖于浏览器的使用，各种各样的网络服务再也不需要像软件一样下载客户端，而是打开就用，数据保存于云端，在哪里都可以无缝使用。(名单来自这里)

Google Chrome

Google Chrome：自从 Google 推出浏览器以来，Chrome 就以势不可挡的趋势占领着市场，超越对手。如果你想拥有一个浏览快速、更新频繁的网络体验，选它没错。

Mozilla Firefox

Mozilla Firefox：一只美丽的狐狸，最早流行于开发者圈子，逐渐被 Google 带向全世界，将 IE 拉下垄断神坛的第一个对手。近来的 Firefox 已经不输于 Chrome 的速度，并且其传统的强大扩展是很多人都无法离开的原因之一。

Opera

Opera：Opera 来自挪威，小众而又精美，只为探索。已经数不清有多少浏览器的新功能诞生自 Opera，早期几乎是手机浏览器的代名词。

Maxthon

Maxthon：源自 MyIE，国人作者创作的浏览器，目前最新版本采用 Webkit（Chrome 使用） 和 Trident（IE 使用） 双排版引擎。

Internet Explorer

Internet Explorer：凭借免费打败 Netscape，凭借 Windows 取得垄断地位，随后不思进取让其树敌过多。如今的 IE 重新在进步，希望能带给我们不断惊喜。

文件资源管理器

资源管理器是 Windows 默认自带的功能简洁，用来管理文件的工具。本着微软出的都有替代品原则，众多优秀的第三方资源管理器出现，给简单又不能随便的文件管理方式带来了革命。(名单来自这里)

Total Commander

Total Commander：几乎无所不能的文件管理器，对非正版用户极其宽容，入门不易，需要认真学习使用的一款佳软。

Clover 3

Clover 3：能为 Windows 资源管理器带来多标签页功能，与操作系统无缝集成，并且还可以将常用文件夹添加为书签，更加快速的在不同文件夹间切换。

Q-Dir

Q-Dir：轻量级的文件管理器，各种布局视图切换灵活，默认四个小窗口组成一个大窗口，操作快捷。软件虽小，粉丝忠诚。

Directory Opus

Directory Opus：相较于 TC，DO 是一款更现代、更适合鼠标操作的文件管理器，功能强大，提高工作效率是其目标。

FreeCommander

FreeCommander：这是一款用来替代 Total Commander 的免费文件管理器，操作体验、功能均靠近 TC。

远程控制工具

远程控制工具 用来远程控制其他电脑，可以辅助他人也可以远程办公，用途很广。(名单来自这里)

Teamviewer

Teamviewer：专业、功能强大，使用简单，对个人用户免费。

QQ 远程协助

QQ 远程协助：集成 QQ 中，可能算作装机量最大的远程控制工具了。

向日葵远程控制

向日葵远程控制：流畅、使用简单，很优秀的国产远程控制工具。

Windows 远程桌面

Windows 远程桌面连接：有时，解决问题的最佳方式是让人向您演示解决方案。

Splashtop

Splashtop：非常不错的商业解决方案，跨平台。
